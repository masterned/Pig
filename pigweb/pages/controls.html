Controls

Basic Play

  1. Select who goes first: Human or Computer
  2. On player's turn, either click roll button to try a roll or hold button to collect round points
  3. On computer's turn, click its roll button
  4. Game ends when player or computer reaches 20 points

Quitting

In order to quit the game, you currently have three primary options. You can close the window. You can press Ctrl+X (Command+X on MacOS). Or, you can select quit in the Game menu at the top of the window via clicking or the associated mnemonic Alt+G+X.

Changing the Computer's Strategy

The computer's play style can be selected at any time during the game as long as it is not currently making a roll. In order to set its strategy, select the desired behavior in the Strategies menu at the top of the window. There are currently 3 different computer strategies.

  1. Cautious - always rolls only once
  2. Random - rolls a random number of times
  3. Greedy - tries to roll 3 times every time

Starting a New Game

If you have completed a game, and you would like to start a new game with the same first player, just press the "Start New Game" button that appears in the center of the window.

If you would like to start a new game without finishing a game you have already started, go to the "Game" menu at the top of the screen and select "New Game". A player must already be chosen for this to work. Otherwise, the option will be grayed-out and will not be clickable. (The button can also be accessed by pressing Ctrl+N (Command+N on MacOS) or Alt+G+N)
